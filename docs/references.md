# 📎 Further reading & inspiration

So you've climbed the educom staircase — from a single **NAND** gate all the way to a computer with a screen and keyboard. 🎉

educom shows you **how** to build a computer. The resources below show you those *same ideas* from other angles: gamified puzzles, friendly videos, real breadboard silicon, university courses, classic books, and — yes — entire CPUs built in Minecraft. Pick whatever matches your mood. There's no wrong door here.

> 🌟 **Brand-new to all this and want the gentlest possible start?** Jump straight to **[Start here](#-brand-new-to-all-this-start-here)** at the bottom.

## 🎮 Learn by playing / interactive

The closest cousins to educom — you *build* things and watch them work.

- **[Turing Complete](https://store.steampowered.com/app/1444480/Turing_Complete/)** (paid game) — Build a CPU from NAND gates, level by level, fully gamified. If you loved educom, this is the natural next obsession. *Beginner-friendly, deeply addictive.*
- **[nandgame](https://nandgame.com/)** (free, in-browser) — Build a whole computer from NAND, one bite-sized level at a time. Nothing to install — open it and start wiring. *The single best free thing to try right now.*
- **[Sebastian Lague — Digital Logic Sim](https://github.com/SebLague/Digital-Logic-Sim)** (free, open source) — A gorgeous sandbox for designing and simulating your own logic circuits. *Great for free-form tinkering once gates click.*
- **[Logisim Evolution](https://github.com/logisim-evolution/logisim-evolution)** (free, open source) — A full-featured digital logic design and simulation tool, used in real classrooms. *More serious; ideal once you want to design bigger circuits.*

## 📺 Video series

Watch someone build the real thing, wire by wire.

- **[Ben Eater — 8-bit breadboard computer](https://www.youtube.com/playlist?list=PLowKtXNTBypGqImE405J2565dvjafglHU)** (playlist) — Builds an actual working computer on breadboards, one chip and wire at a time. The gold standard for "I want to *see* it physically." *Intermediate, endlessly rewatchable.*
- **[Ben Eater — Build a 6502 computer](https://eater.net/6502)** — Wire up a computer around a real, historic 6502 CPU (the chip behind the NES and Apple II). *Intermediate; a fun "real silicon" sequel.*
- **[Crash Course Computer Science](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNlUrzyH5r6jN9ulIgZBpdo)** (playlist) — A friendly, fast-paced survey from logic gates all the way to the modern web. *Beginner-friendly; perfect for the big picture.*
- **[Sebastian Lague — Exploring How Computers Work](https://www.youtube.com/watch?v=QZwneRb-zqA)** — A beautifully animated single video tracing the path from transistors to a running program. *Beginner-friendly, about an hour well spent.*
- **[Branch Education — How are Microchips Made?](https://www.youtube.com/watch?v=dX9CGRZwD-w)** — The literal sand-to-CPU story, with stunning 3D visuals of how a chip is physically manufactured. *Beginner-friendly; pairs perfectly with educom's "it's just sand and NAND" message.*

## 🎓 Courses / curricula

Want a structured, sit-down course? These are the classics.

- **[Nand2Tetris](https://www.nand2tetris.org/)** — The legendary "build a modern computer from first principles" curriculum: gates → CPU → assembler → compiler → OS → a working game. educom is built in exactly this spirit. *The definitive deep dive; free materials.*
- **[Nand2Tetris Part 1 (Coursera)](https://www.coursera.org/learn/build-a-computer)** — The first half (hardware: gates up through the CPU) as a guided online course with deadlines and a community. *Beginner-friendly, structured pacing.*

## 📚 Books

For when you want to curl up and really understand it.

- **[Code: The Hidden Language of Computer Hardware and Software](https://www.charlespetzold.com/code/)** by Charles Petzold — The best gentle on-ramp in existence: it builds a computer from telegraph relays and lightbulbs, starting from *nothing*. **Highly recommended for absolute beginners.** *Easy, delightful.*
- **[The Elements of Computing Systems](https://mitpress.mit.edu/9780262539807/the-elements-of-computing-systems/)** by Nisan & Schocken — The book behind Nand2Tetris. The companion read for the full build-a-computer journey. *Intermediate; hands-on and project-driven.*
- **[But How Do It Know?](http://www.buthowdoitknow.com/)** by J. Clark Scott — A wonderfully approachable, plain-English build-up of a simple CPU, assuming zero background. *Easy; great right after educom's CPU lesson.*
- **Computer Organization and Design** by Patterson & Hennessy — The classic university textbook on how processors and memory really work. *Advanced; the reference you grow into.*

## 🧱 Minecraft / redstone

The "wait — people build *real CPUs* inside a video game?" rabbit hole. (Spoiler: yes, using the exact ideas you built in educom.)

- **[mattbatwings (YouTube)](https://www.youtube.com/@mattbatwings)** — Redstone computers, calculators, and even machine learning built inside Minecraft, with friendly tutorials that teach the logic as he goes. *Beginner-friendly to watch, jaw-dropping to behold.*
- **"Chungus 2" and other community redstone CPUs** — Search for the **Chungus 2** redstone computer (and the broader Minecraft computing community) to see fully functional CPUs — the same NAND-to-CPU ideas from educom — built block by block in a game. *Pure inspiration; proof that what you built scales.*

## 📖 Quick reference (Wikipedia)

Need a crisp explanation of one specific concept? These are solid.

- **[Logic gate](https://en.wikipedia.org/wiki/Logic_gate)** — the building blocks themselves.
- **[NAND logic](https://en.wikipedia.org/wiki/NAND_logic)** — how *everything* is built from NAND (educom's whole premise).
- **[Adder (electronics)](https://en.wikipedia.org/wiki/Adder_(electronics))** — half adders, full adders, ripple carry.
- **[Multiplexer](https://en.wikipedia.org/wiki/Multiplexer)** — the circuit that makes a decision.
- **[Arithmetic logic unit](https://en.wikipedia.org/wiki/Arithmetic_logic_unit)** — the heart of your calculator.
- **[Flip-flop (electronics)](https://en.wikipedia.org/wiki/Flip-flop_(electronics))** — gates that *remember* (the DFF and the clock).
- **[Von Neumann architecture](https://en.wikipedia.org/wiki/Von_Neumann_architecture)** — the CPU + memory blueprint behind the whole machine.

## 🌟 Brand-new to all this? Start here

If you're starting from scratch and just want the gentlest, most welcoming path, pick any of these three:

1. 📚 **[Code: The Hidden Language](https://www.charlespetzold.com/code/)** by Charles Petzold — the friendliest book ever written about how computers work.
2. 📺 **[Crash Course Computer Science](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNlUrzyH5r6jN9ulIgZBpdo)** — short, cheerful videos that paint the whole picture.
3. 🎮 **[nandgame](https://nandgame.com/)** — free and in your browser; build a computer from NAND, just like in educom.

Any one of them will make the next climb feel easy. Have fun out there. ⚡
